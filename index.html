Predicting the Genre of Spotify Songs with Machine Learning

Languages used: Python (scikit-learn, pandas, seaborn, Matplotlib)

Environments used: Jupyter Notebook

This project served as the final assignment for Northeastern University's Foundations of Data Science class for the Summer 2020 semester. I worked with a partner to train popular machine learning algorithms to predict the genre of a song based on select audio features of the track. The audio features, amongst other data, were pulled using Spotify's API and various python libraries, and the project was presented to both the instructors and peers of the class. The project was also my first experience within the field of artificial intelligence.
