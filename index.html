Predicting the Genre of Spotify Songs with Machine Learning

Languages used: Python (scikit-learn, pandas, seaborn, Matplotlib)

Environments used: Jupyter Notebook

This project served as the final assignment for Northeastern University's Foundations of Data Science class for the Summer 2020 semester. I collaborated with a partner to train and test various classification algorithms using key audio features of Spotify songs to classify each song as one of six pre-selected genres. I also used Spotify's Developer API and various python libraries to create a custom dataset featuring various audio features of the Spotify songs.
